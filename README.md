# Hacker Login UI

A minimal, Matrix-style login interface built with pure HTML, CSS, and JavaScript.

## Features
- Matrix rain animation (HTML5 Canvas)
- Particle background (particles.js)
- Neon green hacker-style UI
- Responsive centered layout
- All code in a single HTML file

## How to Run
1. Clone the repo:
   git clone https://github.com/Yuni240/hacker-login.git

2. Open the file:
   xdg-open hacker_login.html

Internet is required for particles.js to load from CDN.

## Live Hosting (GitHub Pages)
Enable via:
Settings → Pages → Branch: main → /root

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API
- particles.js
